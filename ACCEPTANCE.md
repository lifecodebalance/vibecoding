MVP считается готовым, если:
1. можно добавить транзакцию через форму
2. транзакция появляется в списке
3. категория определяется автоматически
4. строится сводка по категориям
5. при превышении лимита появляется предупреждение
6. приложение запускается локально без ручных правок
7. есть минимум 1-2 теста
